Quick Setup

Get Protected in 60 Seconds

Choose your platform and follow the simple steps. No technical expertise required.

1

Open Settings → Network & Internet

Go to your network settings and click on your active connection (Wi-Fi or Ethernet).

2

Edit DNS Settings

Click "Edit" next to DNS server assignment, switch to Manual, and toggle **IPv4 and IPv6** to ON.

3

Enter DNS IPs & DoH Template

Configure both IPv4 and IPv6 to avoid DNS leaks, select **Encrypted only (DNS over HTTPS)**, and enter the templates:

Preferred DNS (IPv4) 
129.153.222.165
Preferred DNS (IPv6) 
2603:c020:26:1f00:0:53ac:f6f0:5281
DNS over HTTPS Template 
https://windows.dnsmalik.duckdns.org/dns-query

Note on Domain: The template domain windows.dnsmalik.duckdns.org is a dual-stack secure alias. It resolves to both our IPv4 and IPv6 addresses. This ensures Windows can establish an encrypted HTTPS connection natively using whichever IP protocol your internet connection prefers.

Want this setting to apply to all Wi-Fi networks and persist permanently? Read the persistence guide
1

Use Native Encrypted DNS Profile

macOS supports native DNS-over-TLS/HTTPS using profiles. Copy this configuration code, save it as `dns.mobileconfig`, and double-click to install it:

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>PayloadContent</key>
    <array>
        <dict>
            <key>DNSSettings</key>
            <dict>
                <key>DNSProtocol</key>
                <string>HTTPS</string>
                <key>ServerURL</key>
                <string>https://mac.dnsmalik.duckdns.org/dns-query</string>
            </dict>
            <key>PayloadDescription</key>
            <string>Configures Secure DNS-over-HTTPS</string>
            <key>PayloadDisplayName</key>
            <string>DNSmalik Secure DoH</string>
            <key>PayloadIdentifier</key>
            <string>com.apple.dnsSettings.managed</string>
            <key>PayloadType</key>
            <string>com.apple.dnsSettings.managed</string>
            <key>PayloadUUID</key>
            <string>8A6B3D6E-C1B0-4A3C-9F4D-9B8A7C6D5E4F</string>
            <key>PayloadVersion</key>
            <integer>1</integer>
        </dict>
    </array>
    <key>PayloadDisplayName</key>
    <string>DNSmalik Secure DNS</string>
    <key>PayloadIdentifier</key>
    <string>org.dnsmalik.secure</string>
    <key>PayloadType</key>
    <string>Configuration</string>
    <key>PayloadUUID</key>
    <string>9F8E7D6C-5B4A-3C2B-1A09-8F7E6D5C4B3A</string>
    <key>PayloadVersion</key>
    <integer>1</integer>
</dict>
</plist>
