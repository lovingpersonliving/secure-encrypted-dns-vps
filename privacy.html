Privacy Commitment

Privacy Policy

We believe privacy is an absolute human right. Below is a detailed breakdown of how we handle request routing.

1. Core Policy: Zero-Logs

DNS MALIK processes DNS lookups entirely in temporary system RAM. We **do not write query logging metrics to physical storage (hard drives/SSD)**, nor do we run background database collections that record client IPs, domain names queried, timestamp stamps, or downstream protocols. Your navigation trail vanishes the millisecond it is completed.

2. Localized Infrastructure Encryption

All client connections to the resolver utilize encrypted transport tunnels, including **DNS-over-HTTPS (DoH)** and **DNS-over-TLS (DoT)**. This ensures that third parties (such as local Wi-Fi providers or upstream internet service providers) cannot perform man-in-the-middle packet sniffing to log your internet requests.

3. Self-Hosted & Non-Commercial

This resolver is hosted strictly on dedicated instances managed under a private family/personal tenancy on Oracle Cloud. Because this is not a commercial enterprise, **we do not sell user traffic details, place telemetry codes, or participate in marketing aggregation networks**.

4. VPN Network Bounds

For added security, our backend resolution interface is restricted to internal virtual network endpoints (WireGuard VPN channels). This prevents scanning entities from logging queries or utilizing the node as an open reflector.

5. Revisions

As this is a private project, any modification to security protocols will be updated directly here on the secure portal page.
